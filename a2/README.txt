This is a sketching program that lets users draw on the screen using a mouse.

The standard drawing features:

    - file menu
    - color and stroke palette to select color and stroke thickness
    - playback controls (slider)

Enhencement:
    - Accurate representation of time. 
      It takes an amount of time to play back a stroke that is proportional to 
      how long it took to draw
    - Customizable color palette
    - ability to draw forward and backward by moving slider
