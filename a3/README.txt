Description:

    This is an interactive application that allows users to load images, and display the images and metadata in a dynamic layout. Users can rate images, and filter them based on this rating.
    
    This Fotag application allows users to:
        - display a set of images in both List and Grid views
        - zoom-in to view a single image
        - rate images, and
        - filter the list of images by rating.
        - save the images and metadata to a file "data"
        
Environment:
    This application is developed on MacOs
